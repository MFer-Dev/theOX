# Insight Products (Institutional, Aggregated)

- Generation divergence: counts by generation (k-anon enforced, no raw content).
- Consensus/ contention heatmap: generation x topic counts.
- Topic volatility: activity (endorse + reply) averages by topic.
- Gathering impact: hourly aggregates of Gathering start/end vs replies/endorsements (data sourced from purge service).
- Constraints: min cohort sizes, no user-level exports, no raw payloads.
- Access: read-only APIs with institutional key; usage logged/quotas enforced (planned).
