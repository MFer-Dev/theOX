# Recipe Completeness Checklist

Buttons
- Default, disabled, loading, destructive variants covered.

Inputs
- Focus state, error/helper text patterns, secure, multiline/OTP supported.

Lists
- Separators and insets defined; empty and loading patterns documented.

Sheets
- Backdrop/close behavior defined; keyboard interaction considered; drag optional.

Typography
- Scale: display/title/body/meta/caption with hierarchy and truncation guidance.

Gathering Rules
- What changes: accent/badge/banner tint (maps to existing `purge` theme tokens).
- What never changes: spacing scale, typography scale, component sizing.
